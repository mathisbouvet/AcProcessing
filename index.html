Welcome to the inventory of gonad processing. The gonads come from human embryos ranging from PCW5.6 to PCW20, with one case from an adult explant. For now, only male gonads are included in the samples. The first section provides an overview of the different antibodies across all developmental stages. By clicking on a cell, you can view detailed information about the combinations used, as well as the antibody details (reference, species, and dilution). When available, a preview image of the individual marker is also shown. This table is interactive and can be freely filtered by structure or antibody. The second table allows for more precise queries with associated combinations. Detailed information is also available for each sample.
